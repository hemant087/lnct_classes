File Handling in Python

1. Introduction to File Handling

File handling in Python allows us to work with files (read, write, append, etc.). Python provides built-in functions to interact with files using the open() function.

2. Opening a File in Python

The open() function is used to open a file. It returns a file object and takes two main arguments:

file_object = open("filename", "mode")

File Modes

Mode Description
"r" Read mode (default), opens a file for reading, gives an error if the file doesn’t exist.
"w" Write mode, creates a new file if it doesn't exist or truncates an existing file.
"a" Append mode, adds data to an existing file without truncating.
"x" Exclusive creation mode, fails if the file already exists.
"b" Binary mode (used with rb, wb, ab, etc.).
"t" Text mode (default), used with rt, wt, etc.

Example

file = open("example.txt", "w") # Opens in write mode file.write("Hello, this is a test file.") # Writing to the file file.close() # Closing the file

3. Reading from a File

You can read a file using different methods:

  • read(size) – Reads the specified number of bytes (default is entire file).
  • readline() – Reads a single line from the file.
  • readlines() – Reads all lines and returns them as a list.

Example: Reading a File

file = open("example.txt", "r") # Open file in read mode content = file.read() # Read entire content print(content) file.close()

4. Writing to a File

Example: Writing Data to a File

file = open("example.txt", "w") # Open in write mode file.write("Python file handling is easy!\n") # Writing a string file.write("We can write multiple lines.\n") file.close()

5. Appending to a File

Example: Appending Data

file = open("example.txt", "a") file.write("Appending a new line.\n") file.close()

6. Using `with` Statement (Context Manager)

Example: Using `with`

with open("example.txt", "r") as file: content = file.read() print(content) # File is automatically closed after the block

7. Checking If a File Exists

import os if os.path.exists("example.txt"): print("File exists") else: print("File not found")

8. File Handling with Exception Handling

try: file = open("nonexistent.txt", "r") content = file.read() file.close() except FileNotFoundError: print("Error: File not found!")

9. Deleting a File

import os os.remove("example.txt")

10. File Handling in JSON

Example: Writing JSON Data

import json data = {"name": "Alice", "age": 25, "city": "New York"} with open("data.json", "w") as file: json.dump(data, file) # Writing JSON to file

11. File Handling in CSV

Example: Writing to a CSV File

import csv data = [["Name", "Age"], ["Alice", 25], ["Bob", 30]] with open("data.csv", "w", newline="") as file: writer = csv.writer(file) writer.writerows(data)

Conclusion

Python provides powerful tools for file handling using open(), read(), write(), and append(). JSON and CSV modules help handle structured data efficiently.
