File Handling in Python

File handling in Python allows you to create, read, write, and manipulate files. It is an essential part of Python programming, especially when working with data stored in files. This guide covers all topics of file handling with examples.

1. Types of Files

a) Text Files

  • Stores data in plain text format.
  • Each line of the file ends with a newline character (\n).
  • Examples: .txt, .csv, .log.

b) Binary Files

  • Stores data in binary format.
  • Examples: Images, videos, .exe files.

2. File Operations

Python provides several built-in functions for file handling:

Function Description
open() Opens a file for reading/writing.
close() Closes the file.
read() Reads the content of the file.
write() Writes content to the file.
readline() Reads a single line from the file.
readlines() Reads all lines as a list of strings.
seek() Changes the file pointer position.
tell() Returns the current file pointer.

3. File Modes

Mode Description
r Read mode. (Default mode)
w Write mode. Creates a new file or truncates.
x Create mode. Fails if the file exists.
a Append mode. Adds data to the file.
b Binary mode. Used with rb, wb, etc.
t Text mode. (Default mode)
+ Read and write mode.

4. Opening and Closing Files

a) Syntax:

file = open("filename", "mode")
# Perform operations
file.close()

Example:

file = open("example.txt", "w")
file.write("Hello, World!")
file.close()

Using with Statement

The with statement automatically closes the file after use.

with open("example.txt", "r") as file:
    content = file.read()
print(content)

5. Reading Files

a) read() Method

Reads the entire content of the file as a single string.

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

b) readline() Method

Reads one line at a time.

with open("example.txt", "r") as file:
    line = file.readline()
    print(line)

c) readlines() Method

Reads all lines and returns them as a list of strings.

with open("example.txt", "r") as file:
    lines = file.readlines()
    print(lines)

Example File Content:

Hello, World!
Python is great.
File handling is easy.

Output for readlines():

['Hello, World!\n', 'Python is great.\n', 'File handling is easy.']

6. Writing to Files

a) write() Method

Writes a string to the file.

with open("example.txt", "w") as file:
    file.write("This is a new file.")

b) writelines() Method

Writes a list of strings to the file.

lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("example.txt", "w") as file:
    file.writelines(lines)

Append Mode (a)

Adds content to the end of the file without overwriting.

with open("example.txt", "a") as file:
    file.write("Appended line.")

7. File Pointer Manipulation

a) tell() Method

Returns the current position of the file pointer.

with open("example.txt", "r") as file:
    file.read(5)
    print(file.tell())  # Output: 5

b) seek() Method

Moves the file pointer to a specific position.

with open("example.txt", "r") as file:
    file.seek(10)
    print(file.read())

8. Handling Binary Files

Reading Binary Files

with open("image.jpg", "rb") as file:
    content = file.read()

Writing Binary Files

with open("new_image.jpg", "wb") as file:
    file.write(content)

9. File Exceptions

Use try-except to handle file-related errors.

try:
    with open("nonexistent.txt", "r") as file:
        content = file.read()
except FileNotFoundError:
    print("File not found.")

10. Examples of File Handling

a) Counting Words in a File

with open("example.txt", "r") as file:
    content = file.read()
    words = content.split()
    print("Word Count:", len(words))

b) Copying a File

with open("source.txt", "r") as source:
    with open("destination.txt", "w") as destination:
        destination.write(source.read())

c) Searching for a Word

with open("example.txt", "r") as file:
    content = file.read()
    if "Python" in content:
        print("Word found!")
    else:
        print("Word not found.")

11. CSV File Handling

Python provides the csv module for handling CSV files.

Reading CSV Files

import csv

with open("data.csv", "r") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Writing CSV Files

import csv

data = [["Name", "Age"], ["Alice", 25], ["Bob", 30]]
with open("data.csv", "w", newline="") as file:
    writer = csv.writer(file)
    writer.writerows(data)

12. JSON File Handling

Python provides the json module for working with JSON files.

Reading JSON Files

import json

with open("data.json", "r") as file:
    data = json.load(file)
    print(data)

Writing JSON Files

import json

data = {"Name": "Alice", "Age": 25}
with open("data.json", "w") as file:
    json.dump(data, file)

13. Best Practices

  • Always close the file using file.close() or with statement.
  • Handle exceptions using try-except.
  • Use relative paths for better portability.
  • Avoid hardcoding file names and use variables or configuration files.
