Assignments : 02

1. Remove Keys with Specific Values

Remove all keys in a dictionary where the values match a specific number.

Test Cases:

Input: {'a': 1, 'b': 3, 'c': 1}, 1

Output: {'b': 3}

Input: {'x': 5, 'y': 10}, 5

Output: {'y': 10}

2. Find Common Keys

Find the common keys in two dictionaries.

Test Cases:

Input: {'a': 1, 'b': 2}, {'b': 2, 'c': 3}

Output: ['b']

Input: {'x': 10, 'y': 20}, {'y': 30}

Output: ['y']

3. Merge Dictionaries

Given two dictionaries, merge them into one. If there are duplicate keys, add their values.

Input:

    dict1 = {'a': 10, 'b': 20, 'c': 30} 
    dict2 = {'b': 5, 'c': 15, 'd': 25}

Output: {'a': 10, 'b': 25, 'c': 45, 'd': 25}

4. Merge Dictionaries

Given two dictionaries, merge them into one. If there are duplicate keys, add their values.

Input:

dict1 = {'a': 10, 'b': 20, 'c': 30} 
dict2 = {'b': 5, 'c': 15, 'd': 25}

Output: {'a': 10, 'b': 25, 'c': 45, 'd': 25}

Input:

dict1 = {'x': 1, 'y': 2}
dict2 = {'y': 3, 'z': 4}

Output: {'x': 1, 'y': 5, 'z': 4}

5. Sort Dictionary by Value

Write a program to sort a dictionary by its values in ascending order.

Input: {'x': 30, 'y': 10, 'z': 20}

Output: {'y': 10, 'z': 20, 'x': 30}

Input: {'a': 3, 'b': 1, 'c': 2}

Output: {'b': 1, 'c': 2, 'a': 3}

6. Group by First Character

Group strings from a list by their first character into a dictionary.

Input: ['apple', 'banana', 'apricot', 'cherry', 'blueberry']

Output: {'a': ['apple', 'apricot'], 'b': ['banana', 'blueberry'], 'c': ['cherry']}

Input: ['cat', 'dog', 'cow', 'duck']

Output: {'c': ['cat', 'cow'], 'd': ['dog', 'duck']}

7. Convert Two Lists into a Dictionary

Write a program to create a dictionary from two lists: one with keys and the other with values.

Input:

keys = ['name', 'age', 'city'] 
values = ['Alice', 25, 'New York']

Output: {'name': 'Alice', 'age': 25, 'city': 'New York'}

Input:

keys = ['id', 'score'] 
values = [101, 95]

Output: {'id': 101, 'score': 95}

8. Find Common Keys with Same Values

Find keys common in two dictionaries that also have the same values.

Input:

dict1 = {'a': 10, 'b': 20, 'c': 30} 
dict2 = {'b': 20, 'c': 25, 'd': 30}

Output: {'b': 20}

Input:

dict1 = {'x': 5, 'y': 10} 
dict2 = {'x': 5, 'z': 15}

Output: {'x': 5}

9. Check Anagram Words

Check if two dictionaries representing letter counts are anagrams of each other.

Test Cases:

Input: {'a': 3, 'b': 2}, {'b': 2, 'a': 3}

Output: True

Input: {'x': 1, 'y': 2}, {'x': 2, 'y': 1}

Output: False

10. Find Keys with Prime Values

Write a program to return all keys whose values are prime numbers.

Test Cases:

Input: {'a': 10, 'b': 3, 'c': 17}

Output: ['b', 'c']

Input: {'x': 4, 'y': 9, 'z': 11}

Output: ['z']

11. Nested Dictionary Sum

Find the sum of all numeric values in a deeply nested dictionary.

Test Cases:

Input: {'a': 5, 'b': {'c': 10, 'd': {'e': 15}}}

Output: 30

Input: {'x': 2, 'y': {'z': 8, 'w': 4}}

Output: 14

12. Sort by Length of Keys

Sort a dictionary based on the length of its keys.

Test Cases:

Input: {'apple': 5, 'a': 3, 'banana': 7}

Output: [('a', 3), ('apple', 5), ('banana', 7)]

Input: {'dog': 2, 'cat': 1, 'elephant': 10}

Output: [('dog', 2), ('cat', 1), ('elephant', 10)]

13. Find Missing Keys

Compare two dictionaries and find keys missing in the second dictionary.

Test Cases:

Input: {'a': 1, 'b': 2}, {'b': 2}

Output: ['a']

Input: {'x': 4, 'y': 5}, {'y': 5}

14. Transpose a Nested Dictionary

Transpose keys and values in a nested dictionary.

Test Cases:

Input: {'a': {'x': 1, 'y': 2}}

Output: {'x': {'a': 1}, 'y': {'a': 2}}

Input: {'p': {'q': 3}}

Output: {'q': {'p': 3}}

15. Count Occurrences of Values

Write a program to count occurrences of values in a dictionary.

Test Cases:

Input: {'a': 1, 'b': 2, 'c': 1}

Output: {1: 2, 2: 1}

Input: {'x': 5, 'y': 5}

Output: {5: 2}

16. Find Top N Highest Values

Write a program to find the top N keys with the highest values in a dictionary.

Test Cases:

Input: {'a': 50, 'b': 20, 'c': 80, 'd': 40}, N = 2

Output: ['c', 'a']

Input: {'x': 15, 'y': 35, 'z': 10}, N = 1

17. Count the Occurrence of a Key

Count how many times a specific key occurs in a list of dictionaries.

Test Cases:

Output: 2

Input: [{'x': 5}, {'y': 10}], 'x'

Output: 1

18. Nested Dictionary Flattening

Flatten a nested dictionary so that the keys are a combination of the nested keys.

Test Cases:

Input: {'a': {'b': {'c': 1}}}

Output: {'a_b_c': 1}

Input: {'x': {'y': 10}, 'z': {'a': 20}}

Output: {'x_y': 10, 'z_a': 20}

19. Key Exists in Nested Dictionaries

Check if a specific key exists in a deeply nested dictionary.

Test Cases:

Input: {'a': {'b': {'c': 5}}}, 'c'

Output: True

Input: {'x': {'y': 10}, 'z': {'a': 20}}, 'b'

Output: False
