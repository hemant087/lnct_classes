Python Dictionary Practice Problems

Problem 1: Add a new key-value pair to a dictionary.
Test Case 1: 
Input: {"name": "John"}, Add: {"age": 30}
Expected Output: {"name": "John", "age": 30}
Test Case 2: 
Input: {"fruit": "apple"}, Add: {"color": "red"}
Expected Output: {"fruit": "apple", "color": "red"}
Problem 2: Remove a key-value pair from a dictionary using `pop()`.
Test Case 1: 
Input: {"brand": "Nike", "product": "Shoes"}, Remove: "product"
Expected Output: {"brand": "Nike"}
Test Case 2: 
Input: {"name": "Alice", "age": 30}, Remove: "age"
Expected Output: {"name": "Alice"}
Problem 3: Merge two dictionaries.
Test Case 1: 
Input: {"a": 1} + {"b": 2}
Expected Output: {"a": 1, "b": 2}
Test Case 2: 
Input: {"x": 10} + {"y": 20}
Expected Output: {"x": 10, "y": 20"}
Problem 4: Count the occurrences of each word in a sentence.
Test Case 1: 
Input: "apple orange apple"
Expected Output: {"apple": 2, "orange": 1}
Test Case 2: 
Input: "cat dog cat bird"
Expected Output: {"cat": 2, "dog": 1, "bird": 1}
Problem 5: Check if a key exists in the dictionary.
Test Case 1: 
Input: {"name": "Alice"}, Check: "name"
Expected Output: Key exists
Test Case 2: 
Input: {"a": 1, "b": 2}, Check: "c"

Problem 6: Get all keys from a dictionary.
Test Case 1:

Test Case 2:

Test Case 1:

Test Case 2:
