Notes on Python Functions

1. Define a Function

Write a function named add_numbers that takes two numbers as arguments and returns their sum.

Test Cases:

  • add_numbers(3, 5) should return 8.
  • add_numbers(-1, 1) should return 0.

2. Function with Default Parameters

Write a function named greet that takes a name and a greeting message. If no greeting is provided, it should default to "Hello".

Test Cases:

  • greet("Alice") should return "Hello, Alice!".
  • greet("Bob", "Good morning") should return "Good morning, Bob!".

3. Return Multiple Values

Write a function named get_min_max that takes a list of numbers and returns the minimum and maximum values.

Test Cases:

  • get_min_max([1, 2, 3, 4, 5]) should return (1, 5).
  • get_min_max([-10, 0, 10]) should return (-10, 10).

4. Function with Variable Arguments

Write a function named calculate_average that takes a variable number of arguments and returns their average.

Test Cases:

  • calculate_average(1, 2, 3, 4) should return 2.5.
  • calculate_average(10, 20, 30) should return 20.0.

5. Check for Prime Number

Write a function named is_prime that checks if a number is prime.

Test Cases:

  • is_prime(5) should return True.
  • is_prime(4) should return False.

6. Factorial of a Number

Write a function named factorial that returns the factorial of a given number.

Test Cases:

  • factorial(5) should return 120.
  • factorial(0) should return 1.

7. Fibonacci Sequence

Write a function named fibonacci that returns the nth Fibonacci number.

Test Cases:

  • fibonacci(5) should return 5.
  • fibonacci(10 ) should return 55.

8. Palindrome Check

Write a function named is_palindrome that checks if a given string is a palindrome.

Test Cases:

  • is_palindrome("racecar") should return True.
  • is_palindrome("hello") should return False.

9. Count Vowels in a String

Write a function named count_vowels that counts the number of vowels in a given string.

Test Cases:

  • count_vowels("hello world") should return 3.
  • count_vowels("Python programming") should return 4.

10. Reverse a String

Write a function named reverse_string that takes a string and returns it reversed.

Test Cases:

  • reverse_string("hello") should return "olleh".
  • reverse_string("Python") should return "nohtyP".
