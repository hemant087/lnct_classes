Python Dictionary Practice Questions

Medium Level Questions

1. Frequency Counter

Write a program to count the frequency of each character in a given string using a dictionary.

Input: "hello world"
Output: {'h': 1, 'e': 1, 'l': 3, 'o': 2, ' ': 1, 'w': 1, 'r': 1, 'd': 1}

2. Merge Dictionaries

Given two dictionaries, merge them into one. If there are duplicate keys, add their values.

Input:
dict1 = {'a': 10, 'b': 20, 'c': 30}
dict2 = {'b': 5, 'c': 15, 'd': 25}
Output: {'a': 10, 'b': 25, 'c': 45, 'd': 25}

3. Sort Dictionary by Value

Write a program to sort a dictionary by its values in ascending order.

Input: {'x': 30, 'y': 10, 'z': 20}
Output: {'y': 10, 'z': 20, 'x': 30}

4. Invert Dictionary

Write a program to invert a dictionary, swapping keys and values.

Input: {'a': 1, 'b': 2, 'c': 3}
Output: {1: 'a', 2: 'b', 3: 'c'}

5. Word Lengths

Create a dictionary from a list of words where the keys are the words, and the values are their lengths.

Input: ['apple', 'banana', 'cherry']
Output: {'apple': 5, 'banana': 6, 'cherry': 6}

6. Nested Dictionary Access

Write a program to access the value of 'math' from the given nested dictionary.

student = {
  'name': 'John',
  'marks': {
    'math': 85,
    'science': 92
  }
}
Output: 85

7. Remove Key with Lowest Value

Write a program to remove the key with the lowest value in a dictionary.

Input: {'a': 10, 'b': 5, 'c': 20}
Output: {'a': 10, 'c': 20}

8. Group by First Character

Group strings from a list by their first character into a dictionary.

Input: ['apple', 'banana', 'apricot', 'cherry', 'blueberry']
Output: {'a': ['apple', 'apricot'], 'b': ['banana', 'blueberry'], 'c': ['cherry']}

9. Convert Two Lists into a Dictionary

Write a program to create a dictionary from two lists: one with keys and the other with values.

Input:
keys = ['name', 'age', 'city']
values = ['Alice', 25, 'New York']
Output: {'name': 'Alice', 'age': 25, 'city': 'New York'}

10. Find Common Keys with Same Values

Find keys common in two dictionaries that also have the same values.

Input:
dict1 = {'a': 10, 'b': 20, 'c': 30}
dict2 = {'b': 20, 'c': 25, 'd': 30}
Output: {'b': 20}
