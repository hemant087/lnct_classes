Map, Filter, and Reduce

1. map Function

The map function applies a function to every item in an iterable and returns a map object.

Syntax:

map(function, iterable)

Examples:

# Squaring Numbers
numbers = [1, 2, 3, 4, 5]
squared = map(lambda x: x ** 2, numbers)
print(list(squared))  # Output: [1, 4, 9, 16, 25]

Example 2: Converting Strings to Uppercase

# Converting Strings to Uppercase
names = ["alice", "bob", "charlie"]
uppercase_names = map(str.upper, names)
print(list(uppercase_names))  # Output: ['ALICE', 'BOB', 'CHARLIE']

Example 3: Adding Corresponding Elements from Two Lists

# Adding Corresponding Elements from Two Lists
list1 = [1, 2, 3]
list2 = [4, 5, 6]
added = map(lambda x, y: x + y, list1, list2)
print(list(added))  # Output: [5, 7, 9]

2. filter Function

The filter function filters elements from an iterable based on a condition.

Syntax:

filter(function, iterable)

Examples:

# Filtering Even Numbers
numbers = [1, 2, 3, 4, 5, 6]
evens = filter(lambda x: x % 2 == 0, numbers)
print(list(evens))  # Output: [2, 4, 6]

Example 2: Filtering Words Longer Than 3 Characters

words = ["hi", "hello", "bye", "world"]
long_words = filter(lambda word: len(word) > 3, words)
print(list(long_words))  # Output: ['hello', 'world']

Example 3: Filtering Positive Numbers

numbers = [-10, -5, 0, 5, 10]
positives = filter(lambda x: x > 0, numbers)
print(list(positives))  # Output: [5, 10]

3. reduce Function

The reduce function applies a function cumulatively to the items of an iterable to reduce it to a single value.

Syntax:

from functools import reduce
reduce(function, iterable[, initializer])

Examples:

# Summing Numbers
from functools import reduce
numbers = [1, 2, 3, 4, 5]
sum_of_numbers = reduce(lambda x, y: x + y, numbers)
print(sum_of_numbers)  # Output: 15

Example 2: Finding the Product of Numbers

from functools import reduce

numbers = [1, 2, 3, 4]
product = reduce(lambda x, y: x * y, numbers)
print(product)  # Output: 24

Example 3: Finding the Largest Number

from functools import reduce

numbers = [3, 7, 2, 9, 5]
largest = reduce(lambda x, y: x if x > y else y, numbers)
print(largest)  # Output: 9

Advanced Examples Combining map, filter, and reduce

Example 1: Sum of Squares of Even Numbers

    from functools import reduce
    
    numbers = [1, 2, 3, 4, 5, 6]
    evens = filter(lambda x: x % 2 == 0, numbers)
    squares = map(lambda x: x ** 2, evens)
    sum_of_squares = reduce(lambda x, y: x + y, squares)
    print(sum_of_squares)  # Output: 56 (2^2 + 4^2 + 6^2)

Example 2: Finding the Longest Word

    from functools import reduce
    
    words = ["python", "java", "c++", "javascript"]
    longest = reduce(lambda x, y: x if len(x) > len(y) else y, words)
    print(longest)  # Output: 'javascript'

Example 3: Transforming and Filtering a List

    numbers = [10, 15, 20, 25, 30]
    transformed = map(lambda x: x / 5, numbers)  # Divide each by 5
    filtered = filter(lambda x: x > 4, transformed)  # Keep results > 4
    print(list(filtered))  # Output: [5.0, 6.0]

Comparison and Use Cases

When to Use Each:

  • map: Transform each element in an iterable (e.g., squaring numbers).
  • filter: Extract elements that satisfy a condition (e.g., finding even numbers).
  • reduce: Aggregate values into a single result (e.g., summing all numbers).

When to Use Each

map:

  • Use when you need to apply the same function to every element in an iterable (e.g., squaring, converting data types).
  • Example: Transform a list of numbers or format strings consistently.

filter:

  • Use when you want to extract elements that satisfy a certain condition.
  • Example: Removing invalid data points or selecting only positive numbers.

reduce:

  • Use when you need to combine all elements of an iterable into a single value (e.g., summing a list, calculating a product).
  • Example: Aggregating numerical results or building a string from a list of characters.
